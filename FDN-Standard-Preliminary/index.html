基石構築標準初探

前言

歡迎來到每系列定番的新牌池標準初探，基石構築（FDN）是個非常與眾不同的系列，相信有許多人到現在還搞不懂這系列的各種產品規劃，像是哪些牌在標準可用？ 哪些牌在補充包內開的到？ 這系列又會在標準合法到甚麼時候？

因此在開始介紹套牌之前，讓我們先來介紹一下這系列本身。

基石構築收錄牌池

基石構築的牌池包含了以下三者：

  • 常規補充包
  • Beginner Box
  • Starter Collection

常規補充包就是我們平常開的補充包，這也是限制賽的總牌池，下面兩個則是額外產品，但也算是FDN系列，所以同樣會進標準。

大原則還是辨認這張卡的系列符號與系列代號（左下角的三字母英文縮寫），只要這張卡的系列符號、代號是FDN，那這張卡就是會進標準的卡。

若想確認完整的FDN收錄牌池，可以參考：FDN全卡表。

基石構築輪替規則

依照目前的賽制公告，FDN的牌直到2029年之前都會在標準合法。

戰鬥階段規則變更

從FDN開始，戰鬥階段迎來一項重大規則變更，原先在宣告阻擋者步驟，宣告阻擋者完成後，如果有聯合阻擋，要立刻 「決定傷害分配順序」 ，現在這一流程已取消，玩家無須宣告任何傷害分配的順序，在傷害結算時可以任意分配傷害，更多戰鬥階段規則細節可以參考文章：回合流程詳解。

套牌介紹

四色門（4 Colors Gates）

套牌連結

迷宮終點回來囉！本套牌來自 約力昂大叔 Uncle Yorin 的分享，由 GF 團隊稍加改動完成。讓興奮探險家爛寶與災禍卜算師格拉布幫助你下一堆地、用拱門天使加一堆血、用掃場穩住場面，然後跑完迷宮出奇致勝吧！
別忘了標準中還有第 11 張門 — 索藍境界通道可以讓你的迷宮走得更加順利，作為一張類似快地的前期站進地，雖然產 mana 需要支付 1 點血做為代價；但在套牌是在三四回開始發動的設計下，能即時站進的門就顯得十分重要了。推薦給喜歡用特殊方式致勝的控制型玩家！

本質上是班特門控，黑色只是為了加入災禍卜算師格拉布、以及撕成碎片的增幅效果。

黑綠妖精滾雪球（Golgari Elvish Snowball）

套牌連結

因應 FDN 強打的妖精種族主題，除了羅堰妖精以外，把標準能放的四種妖精領主們（妖精高位德魯伊、驕傲至美、金葉獵領德維恩、以及 DMU 的葉冠幻術師）全放了，然後再把厲害的傳奇妖精如戮日格莉莎、妖精之刃拉席莉、猛擊英傑泰瓦、好戰不休泰瓦等也放一放，然後因應妖精應該很會產 mana 所以也放了重印的創生浪潮。綠色的部分，旅店主才能是相當契合的，那既然這樣也放一下背叛毒刺瓦絲卡來做個即死 combo 吧！

地牌的部分除了靈魂洞窟滿 4 外，其他還有些許微調的空間；而一費曲線只有四張羅堰妖精仍稍嫌單薄，但只要順利一跳三、三跳無限就會很快滾起致勝的雪球了……吧？

不過，這一版的妖精仍是有以下幾點問題無法在現在的標準牌池很好的被改善：

  1. 一費沒有第二張 mana 頭 — 妖精秘教徒是絕對的硬傷；
  2. 二費沒有善戰妖精這類可以穩定生人的妖精；
  3. 缺乏和聲召集這樣能靈活找出關鍵生物的大招。

這一版的構築就當一個基本的雛型供大家繼續發想，也許未來就能補上這些拼圖了呢！

最後附上一版中速版的黑綠妖精，在一回 mana 頭二回 戮日格莉莎 三回妖精之刃拉席莉或枯葉王侯的進攻節奏下就夠強了！也許把羅堰妖精作為一個綠底中速跳費的發動點，會比在笨重的純妖精思路下表現得更好！

黑綠妖精中速牌表

黑綠惡魔（Golgari Demons）

套牌連結

羅堰妖精將帶動帶綠中速的高歌猛進！原先節奏較差的惡魔體系中速，在獲得一費 mana 頭後將是質的飛躍！試想一回頭、二回開房間、三回深沼翼邪鬼、四回再開五費的房間生一個 6/6 惡魔！即使中間遭受干擾，也會落入惡魔最喜歡的中速節奏，相當建議前標準已開始嘗試惡魔體系的玩家來嘗試！

紅黑惡魔（Rakdos Demon）

套牌連結

惡魔房間體系的另一變體，幫兇魔頭拉鐸司 + 末日拷打魔可以構成即死 combo，再此基礎上嘗試加上九命傭獸，這隻九命怪貓配上拉鐸司也是很搞人，可以反覆攻擊對手牌庫 + 賺牌，還不是那麼好殺，此外因為這惡魔的爆發點節奏比較慢，或許前期也能靠九命怪貓來拖時間。

紅黑蜥蜴（Rakdos Lizard）

套牌連結

套牌實質上變化就是生物多了一個舞火凡爾西諾，但原本的蜥蜴正是缺了一組好的二費生物曲線，所以先前有人會用深窟蝙蝠之類的來填曲線，現在直接喜提一個強力二費曲線，正好蜥蜴最需要的就是對對手造成傷害的能力，不過要注意這隻凡爾西諾不是狂徒，在使用賈斯珀放逐對手牌庫時別數錯數量。

PS：舞火凡爾西諾以前的類別真的叫凡爾西諾，但現在凡爾西諾類別統一被修改為蜥蜴了。

紅綠恐龍（Gruul Dinosaur）

套牌連結

羅堰妖精你來幫依夏理記史人分擔點工作吧，一費 mana 頭只有一組穩定性還有點飄，兩組的話起手上手率就夠高了。

紅藍海盜（Izzet Pirate）

套牌連結

海盜這次也重印了張領主海盜船長， ETB 生珍寶這點也很契合原本海盜的神器配套副主題，不過不可否認的是他本體身材貧弱也缺乏穿透力，實際上到底能不能救起海盜種族的戰力還很難說。

純藍人魚（Mono Blue Merfolk）

套牌連結

雖然這系列人魚的補強都來自重印牌，但這些重印牌就是實用，這套人魚重點強調了 Draw Go 能力，以便最大限度的強化海生割喉客的火力，也讓純藍擅長的康牌比較好發揮。

純紅鬼怪（Mono Red Goblin）

套牌連結

這次鬼怪的待遇也是頗佳，收穫新卡灼勢鬼怪&拋踢擲彈兵，更加強化了鋪場能力，此外暴民頭目克崙可這個明星鬼怪（指揮官）王也回歸了。其實筆者原本還想塞個衝擊震顫強化異次元打擊的能力，但目前實在有點挪不出卡位。

純黑骷髏妖（Mono Black Skeleton）

套牌連結

這次骷髏妖的補強其實很少，但姑且算是把生物曲線補到比較能用的數量了，飾品妙盜細骨有機會讓你在干擾手牌的同時賺一點資源，些許的強化套牌續航力，不過主要還是他是張能用的二費曲線，三費補上重印牌領主死亡爵主，作為正常的種族領主其實沒啥好挑剔了，但要注意他自己的類別是殭屍，只是有幫骷髏妖灌加成的能力而已。

白綠天使（Selesnya Angels）

套牌連結

在 FDN 裡補了大量的天使料，不給這個主題一個篇幅說不過去。但即使希望聖源吉婭妲回歸，但仍無法改變天使部族普遍笨重的設計（又飛又能打設計成低費會太強的這個限制）。剛好羅堰妖精可以很好的補上這塊拼圖，合計八個加速源的部族套牌，在先驅的白綠天使已經印證過可行，那稍微降速的標準中應該也可以嘗試一下！

主牌中的恒曉傳令使與天使命運是為了要更凸顯 FDN 帶來天使部族的補強，組牌時並不一定要採用，可依讀者喜好自行加入喜歡的天使姐姐們！
天使命運真的要使用的話，也許在白底 token 套牌中做一個進攻的突破口會不錯，只要貼成功都可以持續消耗掉對手的解牌、或加速 clock ！

白綠貓（Selesnya Cat）

套牌連結

貓貓也是本系列強推的種族之一，雖然料沒給到像妖精那樣的喪心病狂，但卻印了筆者覺得這次最有誠意的領主：始牙阿拉博，順帶也重印了舊時代的單卡戰神威嚴獰貓，因為領主效應與自己會成長的貓變多了， LCI 的豹族模範庫琪 & 豹王歐奇尼亞豪也會變得好用很多。

黑白蝙蝠（Orzhov Bat）

套牌連結

在原有的蝙蝠架構上加上無限組合技，只要你的場上有星景僧侶或永續頑強體，加上一個嗜血征服者，就能達成無限補血/扣血組合技，而且這些元件即使單獨存在，也跟套牌相性很搭。

紅白家鼠（Boros Mouse）

套牌連結

所以說為何這系列有補血主題、有貓主題，然後阿耶尼不是印獅群之力阿耶尼，而是印了個獅群喚師阿耶尼…？
這家伙長著一臉貓樣但怎麼看都是家鼠專武，那就把它塞進家鼠套牌吧，不管是開 [+1] 還 [-3] 都能觸發英武，另外也塞了一張疾行靴進去，看能不能解決勇行類套牌怕生物被殺的問題。

紅白燒（Boros Burn）

套牌連結

這次一口氣印了閃電波、閃電迸發、波洛斯護符三張高效燒，爆雷煉球雖然是個生物，但本質上也跟一發燒沒什麼差了。（其實你想的話還可以補一個舞火凡爾西諾進去，但現在好牌已經多到不知道怎麼刪了）

以前對抗燒可能用些繫命生物或其他補血源奶一奶，就能撐到燒的子彈耗盡，然而 DSK 印了一張尖鳴宿敵，能夠讓對手整盤遊戲不能補血，而且自己隨便拿張燒點它就能觸發。

老實說筆者真的很懷疑紅白燒會不會超過現在標準的承受程度，跟紅地脈快攻這種上限很高但吃張殺就會當機不同，紅白燒在高效率的同時又很難阻止，而且光是正編打手就能做到對策的對策，雖然這系列也有印執政官威權、水晶屏障這些快攻或燒臉對策牌沒錯，但他們在尊重紅快以外的場合泛用性其實很糟，很難說只為了尊重紅快能否擠出足夠多的卡位。

紅神燒（Ojer Axonil Burn）

套牌連結

上系列尖頭銳器人 + 深邃勇力安雄理尊神的搭配就引起了不小的討論（跟卡價波動），不過真的想以這個為主題組牌時會發現可用配套牌數量又很尷尬，因此總是要混些奇怪的東西進來湊數，然後又把主題變得很稀碎，這次補上舞火凡爾西諾＆閃電迸發算是把合乎主題的牌湊到足夠數量了。

純白補血（Mono White Gain Life）

套牌連結

快攻體系的補血套牌，頻繁的觸發補血可以快速強化阿耶尼的群伴＆菁華通聯師＆明光模範（這傢伙甚至還能賺牌），華輝天使的生天使 token 條件也不能達成，之所以放痛地是為了幫菁華通聯師獲得飛行 + 警戒，至於不放希望地脈則是這張牌有點太 Win More 了，順風局沒他好像也會贏，逆風局根本沒有用，而且我重視的是補血次數，一次補血的量變多影響反而沒那麼大。

這套也是少數可以合理正編執政官威權的套牌，不只幫快攻爭取生物攻防戰時的節奏，也能更加頻繁的觸發補血，有利於強化自己的場面，最重要的是可以給紅快充分的尊重❤️ 因為紅快的尖鳴宿敵只要結算一次異能傷害，這套就整個廢掉了，所以備牌放滿四張濟世翻湧給予充分尊重，不過它在一般對抗去除的對局中當保護咒語也很好用（單論保護能力水晶屏障更強，但跟套牌節奏比較不合）。

PS：內陸祝聖僧雖然是張新卡，但並未收錄在常規補充包牌池內，而是印在起始套組內。

純綠強攻（Mono Green Aggro）

套牌連結

綠色非常正統的大拳頭打法。

純藍節奏（Mono Blue Tempo）

套牌連結

基本上就是上次標準輪替前的藍節奏翻版，因為抉擇，反召喚復刻，重新收穫了一費的濾牌咒語＆能自救的彈生物咒語，暴風雨巨靈目前是被放在備牌作為被剋墳時的Plan B致勝點，正編局要比效率還是原本的狂妄巨靈比較高效，雖然目前牌表是滿編陶拉里亞懼獸，但它跟黯渦蟹是能自由抽換調配率的，兩者各有好壞。

白藍閃現（Azorius Flash）

套牌連結

我是不是最近每系列初探都會寫一次流芳與吉婭妲套牌阿XD

因為重印了魂魅船員，海生割喉客， Draw Go 體系的打點變得更加完整。

白藍飛兵（Azorius Flying）

套牌連結

和上面那套 87% 像，不過強調重點從 Draw Go 干擾能力，變成蒼穹靈鷹的領主膨脹能力。

白藍挖墳（Azorius Reanimator）

套牌連結

在原有的白藍挖墳體系中投入揚潮奇奧拉，主動抽棄可以幫助套牌運轉，自己也是個好的挖墳目標，門檻對這套而言幾乎不是限制。

純白Token（Mono White Token）

套牌連結

用重印的貓貓 token 套裝晶角獸生息所，威嚴獰貓來做為新的 token 供應源，雖然是純白，但因為放了太多功能地，止戈為武卡手率可能有點危險，所以沒採用。

黑白Token（Orzhov Token）

套牌連結

聽說震懼軍將莉蓮娜妳生的是 2/2 殭屍 token 對吧，那就也進來照護者才能 + 永續無瑕體的 token 體系打工吧！

紅黑抽牌殺手（Rakdos Draw Killer）

套牌連結

尖頭銳器人 + 亂塗匍體 + 啟示天災希歐蕊抽牌殺手三劍客大集合，四費曲線的另一個生物之所以選用敵對調查員是希望能用額外的棄牌，去增加對手對抽牌的需求性，另一組三費曲線叛教幫傳道士直接改成難阻刀客也毫無問題，這麼改完後也可以考慮融入惡魔體系，畢竟惡魔體系跟抽牌殺手三劍客差不多各佔一半的生物卡位。

紅黑惡魔契約（Rakdos Demonic Pact）

套牌連結

FDN 預覽時惡魔契約＆無害獻物被放在同一張畫面，大伙看到都笑了出來，這意圖使人幹麻實在太明顯了，雖然這個 combo 是真的可行的致勝手段，但搭配要求還是很硬；套牌設計不建議過度強調靠這個 combo 獲勝，而是把重點放在惡魔契約本身強力的賺資源能力，遊戲計畫還是一般的中速資源戰，不需要考慮怎麼樣才能成功的把跑完三選項的惡魔契約用無害獻物送出去，只要確保自己不會被惡魔契約搞死就好。

與 2016 這個 combo 第一次誕生時不同，現在想要犧牲掉自己的結界方法多到一副牌都塞不完，這裡筆者看上的是 WOE 的加碼（bargain）異能，因為犧牲結界是費用不是效果，不管自己的招有沒有施放成功，至少能確保自己不會被惡魔契約玩死，這裡要特別強調懇求魔鏡的搭配，它不只能夠幫你甩掉惡魔契約，也能吃掉別的肥料來導出無害獻物把惡魔契約送出去，吃掉一個惡魔契約找出新的惡魔契約也是一個玩法，因為懇求魔鏡的存在，我們只要放一張無害獻物就能讓 combo 有足夠的可行性了。至於另一個加碼咒語點燃塔樓更是本來正常紅色套牌就該帶的一費去除，投入套牌根本毫無代價，我們只須確保套牌內有足夠的加碼肥料就好。

藍黑阿芒凱（Dimir Amonkhet）

套牌連結

毫無新意，就是復刻了實用的舊牌，船難探尋客＆黑陽垂暮單卡都符合我們賺 ETB 的遊戲計畫，兩者湊齊時可以變成很惱人的反覆解場 + 回收陣。

藍黑自磨（Dimir Mill）

套牌連結

災沼霸主＆進軍阿芒凱本來就很會堆墳，想要幫助揚潮奇奧拉湊到門檻毫無難度，既然那麼會堆墳，那噁心眼球怪自然也來分一杯羹。

藍黑魁渡（Dimir Kaito）

套牌連結

集合啦！魁渡宗親會！
三隻魁渡都適合搭配高穿透性的藍黑節奏戰生物，這本就是藍黑最擅長的打法，所以可以很自然地融入傳統藍黑節奏的架構中。

話說新的巧忍魁渡其實跟難阻刀客也挺搭的，有了魁渡的 [+1] 就真的是「難阻」刀客了。

紅黑控制（Rakdos Control）

套牌連結

晦澀巨著可以入選筆者最喜歡的標準歷史單卡前 10 名，當年筆者就是仗著晦澀巨著強力的賺牌能力，整出了不少非藍色系控制套牌（當時的最終型態是 Jund Control），不過現在白色搞出了照護者才能 + 永續無瑕體這兩個更加強力的資源引擎，而且吃的解牌幾乎重疊，所以現在想重鑄晦澀巨著榮光的話，會優先挑繞開白色的色組（神器主題另當別論）。

白藍控制（Azorius Control）

套牌連結

上面也提到，因為照護者才能 + 永續無瑕體強力的賺牌能力，近期控制都被 token 體系壟斷了，還是嘗試組組傳統型的白藍 Draw Go 控制看看。但不能否認晦澀巨著論資源跟節奏，確實都追不上那兩個白色結界，唯一的優點就是單卡就能運作，不需要消耗額外的配套卡位。

藍黑控制（Dimir Control）

套牌連結

本體是上系列就存在的至美心智傑斯 + 末日拷打魔 combo 控制（雖然很多時候好像光靠完化傑斯就贏了，惡魔只是有個能偷雞的上限而已），加入晦澀巨著增強資源續航力。

紅白武具（Boros Equipment）

套牌連結

其實說來好笑，本來這套的構築出發點是地脈斧，但組一組後這玩意變成第一個被拔掉的，地脈類效應本來就是個極端放大運氣偏差的東西，而且這東西也不像共振地脈一樣能搞出二回殺的上限，就算起手有拚到地脈斧，等你開始利用它也能也是好幾回合後的事，還不如換回正常的一費武具保持節奏順暢。

話說熱忱冠軍本人 Javier Dominguez 今年又拿下世界冠軍了，如果熱忱冠軍又進標準，那地脈斧就有足夠強力的誘因可以使用了。

蘇勒台鍋子（Sultai Soul Cauldron）

套牌連結

靠著動物群祭師、進軍傳令羅娜、形蹤掠奪靈或其他自磨手段把目標終結者丟到墳場，然後用深淵索命魔造出一個複製品 token，以上都是生物起動式異能，可以用鍋子＆歡騰鬥客泰瓦協助運作。

純藍神器（Mono Blue Artifact）

套牌連結

放飛自我套牌，黯色幻影重印後就會手癢想組個有它的套牌，但傳統 Ramp 套牌現在已經十分完整了，要比加速速度 4 費的黯色幻影肯定比不過，要比資源人家的爬上豆莖抽更爽，所以只能另尋出路了，看來看去好像只有神器主題下才有黯色幻影發揮的空間，起碼吃的到不少神器配合，也有夷城械、非瑞克西亞連境通道這種明確的 Ramp 後終端目標。至於為甚麼不組紅藍或白藍，而是挑了個純藍？這純粹就是我想試試單色的可能性而已，想求保險的話還是推薦組個正常的白藍或紅藍色組，混入低費的優質解牌。
